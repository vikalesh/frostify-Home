# Frostify Rich Home Page - Next.js

Black and orange rich-looking homepage for Frostify Service Center.

## Included

- Next.js App Router
- TypeScript
- Premium black/orange design
- Header with logo, phone and WhatsApp
- Hero with domain input for AC, Washing Machine and Refrigerator
- Why Choose Us with local location and brand expertise
- Location coverage section
- Brand coverage section
- Contact section
- Professional footer
- Third-party disclaimer
- JSON-LD schema
- WhatsApp integration with 98332 00228

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
